## 项目概述

LamMap（前端 + 后端一体）是一个**外贸报价助手**：把客户询盘/需求转成结构化参数与可发送的报价内容，并提供产品库、模板库、附件上传与“邮件附件包（zip）”等能力。后端基于 Gin + Gorm + SQLite，前端基于 Vue3 + Pinia + Element-Plus。

## 主要功能

- **用户系统**
  - 注册/登录（JWT）
  - 获取个人信息
  - 修改密码 / 忘记密码（MVP：不发邮件，仅提示）
- **产品库**
  - 产品 CRUD
  - 批量导入
  - 按参数匹配推荐产品
- **报价**
  - 解析客户需求 → 结构化参数（AI）
  - 生成报价草稿（AI，同步 / 异步 Job）
  - 报价草稿保存/查询/更新/删除
  - 基于历史报价“复制新建”
- **模板库**
  - 多分类模板（quotation/email/chat/confirmation）CRUD
  - 多语言（zh/en）
  - AI 生成模板并保存
- **询盘辅助**
  - 对话生成询盘文本（AI）
  - 按分组提示词批量生成询盘示例（AI）
- **附件**
  - 上传文件到 `/uploads/...`（支持 pdf/word/excel/图片，≤10MB）
  - 附件包列表区分：
    - **上传附件**：有 url，可预览/参与打包
    - **AI 建议附件名**：无 url，默认灰色不可预览；可按条目触发“AI 生成单附件”Job，生成后回填 url
  - **邮件附件包（zip）**：仅打包“勾选且已有 url”的附件，异步生成 zipUrl 并下载

## 技术栈

- **后端**：Go、Gin、Gorm、SQLite（`github.com/glebarez/sqlite`）、Viper（读取 `config.yaml`）、JWT
- **前端**：Vue3（Composition API）、Pinia、Element-Plus、Axios、Vite
- **AI**：OpenAI-compatible `chat/completions`（可配置 Ark 网关）

## 配置与静态文件

- **后端配置**（优先环境变量，其次 `config.yaml`）：
  - `PORT`：默认 8080
  - `DB_PATH`：默认 `quotepro.db`
  - `JWT_SECRET`
  - `OPENAI_BASE_URL`、`OPENAI_API_KEY`、`OPENAI_MODEL`
- **静态目录**
  - `GET /uploads/`*：直接访问后端落盘文件（上传文件、AI 生成 html、zip 等）

## API 约定

- **Base URL**：`/api`
- **认证**：除 `/api/auth/`* 外均需 `Authorization: Bearer <token>`
- **统一响应**（除 middleware.Auth 401 的直出外）：

```json
{ "code": 0, "message": "success", "data": ... }
```

失败：

```json
{ "code": -1, "message": "错误信息", "data": null }
```

## 接口清单（路径 / 功能 / 参数）

### Auth

- **POST** `/api/auth/register`
  - **功能**：注册
  - **Body**：
    - `email` string（必填，email）
    - `password` string（必填，≥6）
    - `name` string（必填）
    - `company` string（可选）
- **POST** `/api/auth/login`
  - **功能**：登录
  - **Body**：`email`、`password`
  - **返回**：`token`、`user`
- **GET** `/api/auth/profile`
  - **功能**：获取当前用户信息
- **PUT** `/api/auth/password`
  - **功能**：修改密码
  - **Body**：`oldPassword`、`newPassword`
- **POST** `/api/auth/forgot-password`
  - **功能**：忘记密码（MVP：不发邮件）
  - **Body**：`email`
- **POST** `/api/auth/reset-password`
  - **功能**：重置密码（MVP：直接按 email 重置）
  - **Body**：`email`、`newPassword`

### Products（产品库）

- **GET** `/api/products`
  - **功能**：列表/搜索/分页
  - **Query**：`search`、`page`（默认1）、`pageSize`（默认20）
- **GET** `/api/products/:id`
  - **功能**：详情
- **POST** `/api/products`
  - **功能**：创建
  - **Body**：Product 字段（由前端传入）
- **PUT** `/api/products/:id`
  - **功能**：更新
  - **Body**：Product 字段
- **DELETE** `/api/products/:id`
  - **功能**：删除
- **POST** `/api/products/import`
  - **功能**：批量导入
  - **Body**：`Product[]`

### 产品匹配

- **POST** `/api/products/match`
  - **功能**：按参数匹配推荐产品（最多10条）
  - **Body**：
    - `productName`、`material`、`size`、`color`、`model`

### Quotes（报价）

- **POST** `/api/quotes/parse`
  - **功能**：AI 解析需求 → 结构化参数
  - **Body（form-data）**：`content` string
  - **Body（json）**：`{ "content": "..." }`（兼容）
  - **返回**：规范化后的 ParsedParams（camelCase，含 `unconfirmed`）
- **POST** `/api/quotes/generate`
  - **功能**：AI 同步生成报价草稿（不建议长任务）
  - **Body（json）**：ParsedParams（可携带 `_replyLanguage` 等 meta 字段）
- **POST** `/api/quotes/generate-jobs`
  - **功能**：创建异步“报价生成”任务
  - **Body（json）**：ParsedParams（可带 meta）
  - **返回**：`jobId`
- **GET** `/api/quotes/generate-jobs/:id`
  - **功能**：查询报价生成任务状态
  - **返回**：`status`、`result`（解析后的对象或 null）、`resultJson`（原始字符串兜底）
- **POST** `/api/quotes`
  - **功能**：保存报价（草稿）
  - **Body（json）**：
    - `customerName/country/currency/deliveryAddress/status/totalAmount/leadTime/remarks/terms/rawRequirement`
    - `parsedParams/replyVersions/confirmationList/attachmentList/templateMeta/renderedContents`
    - `items`（QuoteItem[]）
- **GET** `/api/quotes`
  - **功能**：报价列表（分页/筛选）
  - **Query**：`page`、`pageSize`、`customer`、`status`
- **GET** `/api/quotes/:id`
  - **功能**：报价详情（含 items）
- **PUT** `/api/quotes/:id`
  - **功能**：更新报价（同 CreateQuote 的字段集）
- **DELETE** `/api/quotes/:id`
  - **功能**：删除报价
- **POST** `/api/quotes/:id/duplicate`
  - **功能**：复制报价（创建新草稿）

### 附件（上传/AI生成/打包）

- **POST** `/api/upload`
  - **功能**：通用上传（新建报价页使用）
  - **Body（form-data）**：`file`
  - **返回**：`filename`、`url`（=`/uploads/<YYYY-MM>/<uuid>.<ext>`）
- **POST** `/api/quotes/attachment-generate-jobs`
  - **功能**：AI 生成**单个**附件文件（仅允许 source=ai 且 url 为空；上传附件禁止）
  - **Body（json）**：
    - `params`：ParsedParams
    - `quote`：Quote（用于语言等上下文）
    - `attachment`：`{ name, selected, url:"", source:"ai" }`
  - **返回**：`jobId`
- **GET** `/api/quotes/attachment-generate-jobs/:id`
  - **功能**：查询单附件生成状态
  - **返回**：`attachment: {name,url}`（成功时）
- **POST** `/api/quotes/attachment-zip-jobs`
  - **功能**：把**已存在 url** 的附件打包成 zip（仅打包勾选项）
  - **Body（json）**：`{ attachments: [{name,url,selected,source?}] }`
  - **返回**：`jobId`
- **GET** `/api/quotes/attachment-zip-jobs/:id`
  - **功能**：查询 zip 打包状态
  - **返回**：`zipUrl`（成功时，形如 `/uploads/generated/zips/...zip`）

### Attachments（业务附件记录：可关联产品/报价）

- **GET** `/api/attachments`
  - **Query**：`productId`、`quoteId`
- **POST** `/api/attachments`
  - **Body（form-data）**：`file`，可带 `productId`、`quoteId`
  - **返回**：Attachment 记录
- **DELETE** `/api/attachments/:id`
  - **功能**：删除附件记录，并删除磁盘文件

### Templates（模板库）

- **GET** `/api/templates`
  - **Query**：`category`、`language`
- **GET** `/api/templates/:id`
- **POST** `/api/templates`
  - **Body**：`name/category/language/content`
- **PUT** `/api/templates/:id`
- **DELETE** `/api/templates/:id`
- **POST** `/api/ai/generate-template`
  - **功能**：AI 生成模板并保存
  - **Body**：`category/categoryLabel/language/nameHint/extraHint`

### Dashboard

- **GET** `/api/dashboard/stats`
  - **返回**：今日报价数、待确认数、草稿数、近7天已发送数
- **GET** `/api/dashboard/recent`
  - **返回**：最近 10 条报价（含 items）

### 询盘辅助（AI）

- **POST** `/api/ai/compose-inquiry`
  - **功能**：对话信息 → 生成一段可粘贴询盘文本
  - **Body**：`language`（zh/en）、`answers`（map）
  - **返回**：`content`
- **POST** `/api/ai/generate-inquiry-examples`
  - **功能**：按分组提示词批量生成示例
  - **Body**：`groupName/groupPrompt/language/count(1..10)`
  - **返回**：`examples: [{title,lang,content}]`
