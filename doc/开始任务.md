# 报价回复助手 MVP - 开发任务清单

## 项目技术栈

- **前端**: Vue 3 + Vite + Tailwind CSS + Element Plus + Pinia + Vue Router
- **后端**: Go + Gin + GORM + SQLite
- **AI**: OpenAI API (GPT) 用于需求解析和内容生成

- [ ] 初始化 Go 后端项目 (Gin + GORM + SQLite)

## 三、后端开发任务

### 3.1 项目基础

- [ ] Gin 路由配置

- [ ] 统一响应格式
- [ ] 配置文件管理

### 3.2 数据库模型

- [ ] User 用户表
- [ ] Product 产品表
- [ ] Quote 报价表
- [ ] QuoteItem 报价明细表
- [ ] Template 模板表
- [ ] Attachment 附件表

### 3.3 认证模块 `/api/auth`

- [ ] POST `/api/auth/register` 注册
- [ ] POST `/api/auth/login` 登录 (返回 JWT)
- [ ] GET `/api/auth/profile` 获取当前用户信息
- [ ] 密码加密 (bcrypt)
- [ ] JWT Token 生成与验证

### 3.4 产品模块 `/api/products`

- [ ] GET `/api/products` 产品列表 (分页/搜索/筛选)
- [ ] GET `/api/products/:id` 产品详情
- [ ] POST `/api/products` 新建产品
- [ ] PUT `/api/products/:id` 更新产品
- [ ] DELETE `/api/products/:id` 删除产品
- [ ] POST `/api/products/import` Excel 批量导入

### 3.5 报价模块 `/api/quotes`

- [ ] POST `/api/quotes/parse` AI 解析客户需求 → 结构化参数
- [ ] POST `/api/quotes/generate` 生成报价单 + 话术 + 确认清单
- [ ] POST `/api/quotes` 保存报价
- [ ] GET `/api/quotes` 报价列表 (分页/筛选)
- [ ] GET `/api/quotes/:id` 报价详情
- [ ] PUT `/api/quotes/:id` 更新报价
- [ ] DELETE `/api/quotes/:id` 删除报价
- [ ] POST `/api/quotes/:id/duplicate` 基于旧报价新建
- [ ] GET `/api/quotes/:id/export/pdf` 导出 PDF
- [ ] GET `/api/quotes/:id/export/excel` 导出 Excel

### 3.6 模板模块 `/api/templates`

- [ ] GET `/api/templates` 模板列表
